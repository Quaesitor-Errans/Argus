Argus
Mission
Argus is an explainable information-space analysis platform.
Its purpose is not to summarize news.
Its purpose is to measure, compare and explain how information evolves across sources, countries and time.

Core principles
Everything must be explainable.
Every conclusion must reference measurable evidence.

Measurements before conclusions.
Argus stores observations, not opinions.

Events are primary.
Articles are evidence.

Claims are atomic.
Every statement should be decomposed into verifiable claims.

Truth is evidence-based.
No statement should be accepted solely because it comes from a particular source.

Every hypothesis has confidence.
There are no absolute conclusions.

Every result is reproducible.
Running the same analysis on the same data should produce the same result.

No black boxes.
Every score must be decomposable into its components.
